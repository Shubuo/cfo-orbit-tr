# Katki Rehberi

Bu proje Turk pazarina odakli, cok‑ajanli bir Personal CFO sistemidir. PR'leri memnuniyetle karsiliyoruz.

## Gelistirme Akisi
1. Fork alin ve yeni bir branch olusturun (`feat/...` veya `fix/...`).
2. Degisikliklerinizi kucuk ve hedefli tutun.
3. `python main.py` ile temel akisi dogrulayin.
4. PR acin ve acik bir aciklama yazin.

## Kod Standartlari
- Python type hint kullanimina devam edin.
- Yeni fonksiyonlar icin kisa ve faydali yorum ekleyin.
- Turkce yalnizca rapor ve terminal ciktilarinda kullanilmalidir.

## Issue & PR Notlari
- Hatali veri kaynaklari veya API degisimleri icin issue acin.
- Yeni veri kaynaklari ekleyecekseniz `tools.py` icinde izole edin.
